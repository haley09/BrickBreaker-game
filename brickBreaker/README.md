# Brick Breaker Game

A desktop Brick Breaker game built with Java Swing. The project includes paddle movement, ball physics, brick collision detection, score tracking, a lives system, and restart functionality.

## Technologies
- Java
- Swing
- AWT Graphics

## Features
- Real-time game loop using Timer
- Paddle and ball collision handling
- Brick destruction and score tracking
- Lives system
- Restart support
- Start menu screen
